# FixMzansi MVP Scope

## Goal
Build the smallest product that proves a homeowner can find, trust, and book a tradesperson faster and more reliably than WhatsApp referrals.

## Core user problem
South Africans already spend money on home services, but the current process is fragmented:
- ask around in WhatsApp groups
- compare random referrals
- call multiple people
- negotiate by phone
- pay manually
- hope the work is good

## MVP promise
"Trusted trades, instantly booked, fairly priced."

## MVP target market
Start with one metro and one urgent trade category.
Recommended wedge:
- City: Johannesburg / Pretoria
- Category: plumbers first, then electricians

## MVP must include
1. Consumer request flow
   - service type
   - job description
   - photos
   - suburb / location
   - urgency

2. Pro directory
   - verified profile
   - service area
   - rating
   - completed jobs
   - contact / booking button

3. Matching
   - show 3 to 5 relevant pros
   - rank by proximity, availability, rating, verification status

4. Booking
   - request a time slot
   - confirm via web
   - notify pro and consumer

5. Trust layer
   - trade certificate upload
   - manual vetting at first
   - verified badge
   - completed-job reviews only

6. Basic payments
   - start with reservation / deposit flow if full escrow is too heavy
   - capture the intent to pay digitally from day one

7. Admin tools
   - add / approve pros
   - review jobs
   - resolve disputes
   - edit listings

## Explicitly out of scope for MVP
- full BNPL
- mobile app
- insurance / warranties
- complex AI pricing engine
- multi-city launch
- large B2B dashboard
- advanced automation

## MVP success criteria
The MVP is successful if:
- users can find a pro in under 2 minutes
- pros can receive bookings without friction
- at least some jobs complete successfully
- users trust the platform enough to come back
- the product can be expanded city by city

## MVP principle
If a feature does not improve trust, booking speed, or conversion, it does not belong in the MVP.
