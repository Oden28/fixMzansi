# FixMzansi

Cape Town solar marketplace for trusted trades and verified pros.

FixMzansi helps homeowners and small businesses request solar work, get matched with verified professionals, book jobs, track progress, and leave reviews.

## Source of truth
The canonical product and build docs live in `docs/`.

## Current shape
- Marketplace discovery pages use seeded data where needed
- Request, booking, notification, and admin flows are wired to Supabase
- The codebase is being tightened so the product reads as one coherent marketplace end to end
