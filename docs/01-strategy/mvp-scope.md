# MVP Scope

FixMzansi MVP = a Cape Town solar marketplace that can reliably move a customer from intent to completed job.

MVP must support:
- Customer request creation
- Pro discovery and request matching
- Pro directory and pro profile pages
- Booking creation and booking status updates
- Customer dashboard and provider dashboard views
- Notification inbox
- Admin moderation queue for pros, requests, and operational items
- Review submission after completed work

Out of scope for MVP:
- Full payments integration
- Multi-city expansion
- Complex quoting workflows
- Advanced analytics
- Native mobile apps

MVP success means the product feels coherent end to end:
request -> match -> book -> complete -> review
