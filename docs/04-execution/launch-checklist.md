# Launch Checklist

## Product readiness
- Request form works end to end
- Matching renders meaningful providers
- Booking pages explain status clearly
- Review flow only appears at the right time
- Notification inbox reflects actual read state
- Admin console matches the real schema

## Technical readiness
- Supabase environment variables configured
- Database migrations applied
- Seed data loaded for demo mode
- Build passes locally
- Tests pass

## Operational readiness
- Clear onboarding copy
- Verified providers visible in the directory
- Support path defined for broken requests or bookings
- No placeholder language on critical marketplace screens
