# FixMzansi Docs

This folder is the source of truth for the product and build.

FixMzansi is a Cape Town solar marketplace that helps homeowners and small businesses request work, get matched with verified pros, book jobs, track progress, and leave reviews.

Current build shape:
- Product and architecture docs describe the intended marketplace
- The app already contains request, matching, booking, dashboard, notifications, and admin surfaces
- Discovery content is partly seeded for presentation, while the request/booking/admin plumbing is wired to Supabase

Doc map:
- 00-overview — product summary and positioning
- 01-strategy — market, scope, moat, and revenue model
- 02-product — user journeys, feature set, and UI/product spec
- 03-architecture — implementation plan and technical architecture
- 04-execution — launch checklist and rollout plan
